

## 什么是链表

链表是由一组不必相连(不必相连：可以连续也可以不连续)的内存结构 (节点)，按特定的顺序链接在一起的抽象数据类型。

## 常见链表

链表常用的有 3 类：

### 单链表

由各个内存结构通过一个 `Next` 指针链接在一起组成，每一个内存结构都存在后继内存结构【链尾除外】，内存结构由数据域和 `Next` 指针域组成。

### 双向链表

由各个内存结构通过指针 `Next` 和指针 `Prev` 链接在一起组成，每一个内存结构都存在前驱内存结构和后继内存结构【链头没有前驱，链尾没有后继】，内存结构由数据域、`Prev` 指针域和 `Next` 指针域组成。

### 循环链表

#### 单向循环链表

由各个内存结构通过一个指针 `Next` 链接在一起组成，每一个内存结构都存在后继内存结构，内存结构由数据域和 `Next` 指针域组成。

#### 双向循环链表

由各个内存结构通过指针 `Next` 和指针 `Prev` 链接在一起组成，每一个内存结构都存在前驱内存结构和后继内存结构，内存结构由数据域、`Prev` 指针域和 `Next` 指针域组成。

### 链表中环的检测

```java
  /**
     * 判断链表是否有环 采用快慢指针
     * 若有环，快指针先行进环，在环中绕圈，慢指针后入环，也在环中绕圈，
     * 由于快指针每次比慢指针多走一步，意味着两个指针在环中的位置每次都缩短一步，
     * 所以若有环，两个指针在环中必能相遇
     * 如: 1 - 2 - 3 - 4 -2  如此一个环
     * 慢指针遍历:1 2 3
     * 快指针遍历:2 4 3
     * 此时 快慢指针相遇  那么证明 存在环,返回true
     *
     * @param head
     * @return
     */
    private boolean hasCycle(Node head) {
        if (head == null) {
            return false;
        }
        Node slow = head;
        Node fast = head.next;
        while (fast.next != null && fast != null) {
            if (slow == fast) {
                return true;
            }
            slow = slow.next;
            fast = fast.next.next;
        }
        return false;
    }

    @Test
    public void testHasCycle() {
        Node head = new Node('a');
        head.next = new Node('b');
        head.next.next = new Node('c');
        head.next.next.next = new Node('d');

        System.out.println(hasCycle(head));

        head.next.next.next.next = head.next;

        System.out.println(hasCycle(head));
    }
```

### 两个有序的链表合并

```java
 /**
     * 两个有序的链表合并
     * 解题思路:
     * 因为有序 所以可以挨个比较
     * 借用临时变量,将新的链表组合出来
     *
     * @param l1
     * @param l2
     * @return
     */
    private Node mergeTwoLists(Node l1, Node l2) {
        Node root = new Node(null);
        Node temp = root;
        while (l1 != null && l2 != null) {
            if (l1.value > l2.value) {
                temp.next = l2;
                l2 = l2.next;
            } else {
                temp.next = l1;
                l1 = l1.next;
            }
            temp = temp.next;
        }
        return root.next;
    }

        l2.next = new Node('d');
        l2.next.next = new Node('f');

        Node node = mergeTwoLists(l1, l2);

    }
```

### 删除链表倒数第n个结点

```java
 /**
     * 删除链表倒数第n个结点
     * 解题思路:
     * 利用双指针
     * 倒数第x,可以通过如下方式理解
     * 假设 当前链表 拥有n个 节点.
     * 倒数第x  就是 n - x
     * 那么  我现在定义两个指针, 慢指针 指向 1  和  快指针指向 x
     * 快慢指针  每次前进一步. 当 快指针 指向n 时,慢指针就指向了 n-x 位置
     * 已知删除一个节点需要知道该节点的上一个节点.
     * 这个时候 增加root 节点, 一举两得,既可以保证 快慢双指针 都指向上一个节点,有保证了第一个节点的删除
     * 最后返回 root 节点的下一个  及时 新的链表
     *
     * @param head
     * @param n
     * @return
     */
    private Node removeNthFromEnd(Node head, int n) {
        if (n == 0 || head == null) {
            return head;
        }
        Node root = new Node(null);
        root.next = head;
        Node fast = root;
        Node slow = root;
        for (int i = 0; i < n + 1; i++) {
            fast = fast.next;
        }
        while (fast != null) {
            fast = fast.next;
            slow = slow.next;
        }

        slow.next = slow.next.next;
        return root.next;

    }

    @Test
    public void testRemoveNthFromEnd() {
        Node root = creatNode("123456789");
        Node node = removeNthFromEnd(root, 9);
        printList(node);

    }
```



### 求链表的中间结点 

```java
	  /**
     * 求链表的中间结点 如果链表长度为偶数 那么取后一位
     * 解题思路:
     * 双指针
     *
     * @param head
     * @return
     */
    private Node middleNode(Node head) {
        Node fast = head;
        Node slow = head;
        while (fast != null && fast.next != null) {
            fast = fast.next.next;
            slow = slow.next;
        }
        return slow;
    }
```
