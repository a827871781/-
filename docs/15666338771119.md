## 截图

| 组合键                  | 功能                                                   |
| ----------------------- | ------------------------------------------------------ |
| Command-Shift-3         | 将屏幕捕捉到文件                                       |
| Command-Shift-Control-3 | 将屏幕内容捕捉到剪贴板                                 |
| Command-Shift-4         | 将所选屏幕内容捕捉到一个文件，或按空格键仅捕捉一个窗口 |
|Command-Shift-Control-4 | 将所选屏幕内容捕捉到剪贴板，或按空格键仅捕捉一个窗口|
