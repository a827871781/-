## 自定义快捷键

| 快捷键          | 功能                                 |
| --------------- | ------------------------------------ |
| ctrl + j        | 向上一行                             |
| ctrl + k        | 向下一行                             |
| ctrl + h        | 向左移动一个单词                     |
| ctrl + l        | 向右移动一个单词                     |
| alt + k         | 跳转下一个方法                       |
| alt + j         | 跳转上一个方法                       |
| ctrl + alt + j  | 选中当前方法内，光标处单词，统一修改 |
| ctrl + ；       | emacsIDEAs 搜索 单词快捷键           |
| ctrl + alt + h  | 查看当前类结构                       |
| shift + esc     | 关闭弹框                             |
| alt + g         | 当前文件顶                           |
| alt + shift + g | 当前文件底                           |
